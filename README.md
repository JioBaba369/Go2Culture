
# Go2Culture

Go2Culture is a web application that connects travelers with local hosts for authentic cultural and dining experiences. It's a platform where hosts can share their culture, and guests can discover unique activities like in-home dining, cooking classes, and more.

This project was built with [Firebase Studio](https://firebase.google.com/studio).

## Core Features

-   **Host Onboarding**: Streamlined process for hosts to create profiles and list their cultural experiences.
-   **Experience Discovery**: Guests can search and filter experiences by location, category, and other criteria.
-   **Secure Bookings**: Secure online payments and booking management for both hosts and guests, including coupon support.
-   **Host-Guest Communication**: Direct messaging between hosts and guests to coordinate details.
-   **Rating & Review System**: A review system to build trust and ensure quality experiences.
-   **Admin Dashboards**: A comprehensive set of dashboards for administrators to manage users, applications, experiences, bookings, and more.
-   **Host Dashboards**: A dedicated dashboard for hosts to manage their listings, view bookings, and set their availability.

## Tech Stack

-   **Framework**: [Next.js](https://nextjs.org/) (with App Router)
-   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
-   **UI Components**: [ShadCN/UI](https://ui.shadcn.com/)
-   **Backend & Database**: [Firebase](https://firebase.google.com/) (Authentication, Firestore)
-   **Form Management**: [React Hook Form](https://react-hook-form.com/) with [Zod](https://zod.dev/) for validation
-   **Deployment**: [Firebase App Hosting](https://firebase.google.com/docs/app-hosting)

## Getting Started

### Prerequisites

-   [Node.js](https://nodejs.org/) (v20 or later)
-   npm

### Running the Development Server

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

The application will be available at `http://localhost:9002`.
