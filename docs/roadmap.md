# Go2Culture: Development Roadmap

This document outlines the development plan for Go2Culture, focusing on key features and milestones. Our goal is to build a trusted, human-centric platform for authentic cultural experiences.

## Phase 1: Foundation & Core Guest Experience (Current State)

This phase establishes the essential features for users to discover and learn about experiences.

-   [x] **User Authentication:** Secure sign-up, login, and password reset.
-   [x] **Static Pages:** About, Contact, Privacy, Terms, etc.
-   [x] **Experience Discovery:** Homepage with featured content and a `/discover` page with basic filtering.
-   [x] **Experience Details Page:** A detailed view for each experience.
-   [x] **Public User Profiles:** View-only profiles for hosts.

## Phase 2: Booking & Host Interaction

This phase focuses on enabling the core transaction and communication loop between guests and hosts.

-   [ ] **Secure Booking System:** Implement the booking widget on the experience page, allowing guests to select dates and number of guests.
-   [ ] **Wishlist Functionality:** Allow authenticated users to save experiences.
-   [ ] **Messaging System:** A booking-centric chat for guests and hosts to communicate.
-   [ ] **Guest Profile Management:** Allow users to edit their own profiles.

## Phase 3: Host Lifecycle & Community Trust

This phase empowers hosts and builds trust within the community.

-   [ ] **Host Onboarding:** A multi-step application flow for users to become hosts.
-   [ ] **Host Dashboard:** A central portal for hosts to manage their listings, bookings, and availability.
-   [ ] **Rating & Review System:** Allow guests to leave reviews after an experience is completed.
-   [ ] **Host Calendar Management:** A visual interface for hosts to block and unblock dates.

## Phase 4: Administration & Platform Scaling

This phase introduces the tools needed for platform management and growth.

-   [ ] **Admin Dashboard:** A comprehensive back-office for platform oversight.
-   [ ] **Host Application Review:** An admin interface to approve, reject, or request changes on host applications.
-   [ ] **Platform-wide Management:** Admin views for managing all users, experiences, bookings, and reports.
-   [ ] **Promotional Tools:** Admin system to create and manage coupons and sponsors.
